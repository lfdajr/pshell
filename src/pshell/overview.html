Pleno Shell (pshell)

O pshell é um aplicativo de utilização simples, e que pode ser expandido para facilitar a automação de tarefas. Todos os comandos do pshell são métodos de classes Java que podem ser acionados a partir da interface-padrão do aplicativo. Qualquer classe Java e seus métodos poderão ser chamados via pshell.

Iniciando o pshell

Para inicializar o pshell abra uma linha de comando e digite:
    java -jar PlenoShell.jar
Quando o pshell iniciar, o usuário perceberá uma linha de comando disponível. Nesta linha de comando, existe a informação sobre a classe Java que está sendo utilizada no momento. Digite:
    versao
O usuário perceberá que foi exibida na tela a versão que está sendo utilizada do pshell. Veja a classe pshell.base.ShellBaseClass para saber os principais comandos do pshell.

Formato dos comandos pshell

Os comandos pshell devem possuir um dos seguintes formatos:
  • comando
  • comando arg1 arg2 arg3 ...
Para alguns comandos, set por exemplo, a separação por espaço em branco entre os argumentos poderá não ser adequada. Neste caso, o pshell fornece uma maneira ao usuário para que informe ao aplicativo que aquela seqüência de caracteres faz parte de apenas um argumento. Ex.:
  • set load [use pshell.base.ShellDBBaseClass]
